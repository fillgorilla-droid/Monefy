# Monefy

Monefy is a console-based personal finance management system written in C++. The project was created as a learning exercise to practice object-oriented programming, dynamic memory management, file handling, and working with custom data structures.

The application allows users to keep track of their personal finances by recording income and expenses, organizing them into categories, setting budget limits, and viewing financial statistics.

## Features

- Add and manage income
- Add and manage expenses
- Create and edit categories and subcategories
- Set budget limits for different categories
- View current balance
- Display financial statistics for a selected period
- Create, edit, and remove reminders
- Save and load all data from a binary file
- Console-based menu system

## Technologies

- C++
- Object-Oriented Programming
- Dynamic memory allocation
- File I/O
- Custom `MyString` class

## Purpose

The main goal of this project was to improve C++ programming skills by building a complete console application from scratch without using the STL string class. During development, special attention was given to class design, memory management, working with files, and implementing a structured menu-driven interface.
